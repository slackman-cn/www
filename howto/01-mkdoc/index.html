MkDocs 创建网站

https://www.mkdocs.org/getting-started/

sudo apt install python3-venv
sudo apt install python3-pip

python3 -m venv mysite
source bin/activate
pip config set global.index-url https://mirrors.tuna.tsinghua.edu.cn/pypi/web/simple
pip install mkdocs

mkdocs new website
mkdocs serve
