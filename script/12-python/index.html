算法3: Fib数列

def fib(n):
    if n == 1 or n == 2 :
        return 1
    return fib(n-1) + fib(n-2)

检查 0,1,1,2,3,5,8,13 ...
https://oi-wiki.org/math/combinatorics/fibonacci/

==== 方式1

b = 1
for n in range(3,41):
    tmp = a+b

    b = tmp
    print(f'fib({n}) = {b}')
